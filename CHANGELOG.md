# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-01

### Added

- Initial release
- Full coverage of the Dubbl API v1
- Synchronous and asynchronous clients
- Automatic retry with exponential backoff
- Typed exception hierarchy
- Support for all API resources: accounts, entries, invoices, bills, contacts, payments, bank accounts, bank transactions, quotes, expenses, inventory, organization, members, API keys, reports, audit log, tax rates, budgets, fixed assets, projects, cost centers, credit notes, purchase orders, recurring transactions, fiscal years, webhooks, roles, tags, documents, attachments, trash, export, and backups
